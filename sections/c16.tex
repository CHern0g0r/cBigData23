\section{Apache Spark: трансформация и действие (примеры), lineage.}

\D{
    Трансформация. Не приводит к запуску вычислений. Описывает способ
    получения новой RDD из старой.
    \begin{itemize}
        \item map(f)
        \item filter(f)
        \item union(otherDataset)
        \item reduceByKey(func)
        \item join(otherDataset)
    \end{itemize}
}

\D{
    Действие. Запускает цепочку вычислений. Предполагают запись результата в
    хранилище.
    \begin{itemize}
        \item collect()
        \item count()
        \item take(n)
        \item saveAsTextFile(path)
    \end{itemize}
}

\D{
    Lineage отслеживает процесс выполнения задачи и помогает восстановить
    после падения.

    Отслеживает путь в графе, необходимый для получения потерянных
    данных.
}
